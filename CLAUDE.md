# SmartPlan

Инструмент проектирования систем умного дома и видеонаблюдения.

## Стек

- Electron + React + Vite + TypeScript
- Версия: 3.1.0

## Команды

```bash
npm run dev
npm run build
npm run dist
```

## Важно

- GitHub: `laskinss27-cmyk/smartplan`
- Дистрибутив: `SmartPlan-{version}-win.zip`
- Подхватывается лаунчером автоматически
